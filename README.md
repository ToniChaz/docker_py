# Docker example

A docker example with python, redis and some HTML5 forn end application.

  - Docker
  - Python
  - Redis
  - HTML5

### Version
1.0.0

### Tech

### Installation
_you need have installed docker_

```
cd docker_py
docker-compose up
```

Your application is running on http://localhost:5000
